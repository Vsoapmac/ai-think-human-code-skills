# AI 思考，你写代码

## 核心理念

**不是"为了独立思考和编程能力而放弃 AI"，而是"享受 AI 带来的便利的同时，用约束来保护你的核心竞争力"。**

你可以随心所欲地使用 AI——让它扫描项目、分析需求、评估风险、提供创新方案。但最终敲键盘的人是你自己决定。该 skill 像一个隐形的护栏，让你在高速飞驰时不至于失去对方向的掌控。

## 它做什么

当你向 AI 提出编码请求时，该 skill 会自动介入：

1. AI 先扫描项目上下文，了解代码结构和依赖
2. 分析问题难度、重复度和安全风险
3. 提供 2-3 种解决思路和关键代码片段
4. 然后问你一句：**"你来写还是我来写？"**

两种模式自由切换：

- **你来写（引导模式）**：AI 只给思路和片段参考，你亲手写，写完 AI 审查
- **你来写（教学展示模式）**：AI 直接写完整代码 + 原理拆解(理解后你可以选择自己重写一遍)

## 适用场景

- 使用 AI 辅助编程但仍想保持代码掌控感
- 担心长期依赖 AI 导致编程思维退化
- 想利用 AI 的创新解法，但不想只是复制粘贴
- 涉及安全问题需要对 AI 的代码保持警惕

## 安装

将 `ai-think-human-code/SKILL.md` 放置到 opencode 的 skills 目录：

```
~/.config/opencode/skills/ai-think-human-code/SKILL.md
```

重启 opencode 后生效。无需额外配置。
